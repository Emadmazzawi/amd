# VLSI Verification Co-Pilot

An AI-powered assistant designed for hardware engineers to optimize Verilog code, identify synthesis risks, and debug testbenches. Fine-tuned on AMD MI300X GPUs using ROCm.

## Project Structure
- `dataset/`: Contains training data for fine-tuning.
- `scripts/`: Training and utility scripts.
- `app/`: Streamlit web application.
- `triton/`: Reference materials for Triton/ROCm programming.
